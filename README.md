# Code-Alpha-Task-2-Foody-Express-Wireframing-High-Fidelity-Design

**Figma Link:** https://www.figma.com/design/EW7dvHOGSqvqwIBLrNQgya/Food-Delivery-App-High-Fidelity-Design?node-id=0-1&t=P3jRw3TYYfshKUxS-1 

**Overview**: Foody Express is a high-fidelity mobile application design created to provide users with a seamless and visually engaging food ordering experience. The project transforms the initial wireframes into a polished user interface by incorporating colors, typography, icons, imagery, and modern UI components. The application allows users to browse restaurants, explore menu items, manage their cart, complete secure checkouts, and track orders through a clean and intuitive interface.

**Tools Used**
1. **Figma** – UI Design and Prototyping
2. **Microsoft Word** – Project Documentation
3. **PDF** – Final Design Report

**Features**
1. User Login and Registration
2. Restaurant Discovery
3. Food Category Browsing
4. Menu Exploration
5. Add-to-Cart Functionality
6. Order Summary and Checkout
7. Order Confirmation and Tracking

**Design Highlights**
1. Clean and modern mobile interface
2. Consistent visual hierarchy
3. User-friendly navigation
4. High-quality food imagery
5. Organized information architecture
6. Simplified ordering workflow

**Screens Included**
1. Login & Authentication
2. Home Dashboard
3. Restaurant Menu
4. Cart Summary
5. Checkout
6. Order Confirmation

**Outcome:** The final design provides an intuitive and visually appealing food delivery experience, focusing on usability, accessibility, and efficient user interaction. The project demonstrates the complete user journey from account login to successful order placement.
