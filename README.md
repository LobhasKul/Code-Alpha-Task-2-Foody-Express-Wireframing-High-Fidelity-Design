# Code-Alpha-Task-2-Foody-Express-Wireframing-High-Fidelity-Design

**Figma Design Link:** https://www.figma.com/design/EW7dvHOGSqvqwIBLrNQgya/Food-Delivery-App-High-Fidelity-Design?node-id=0-1&t=P3jRw3TYYfshKUxS-1

**Overview:** Foody Express is a high-fidelity mobile food delivery application designed to provide users with a seamless and visually engaging food ordering experience. The project transforms low-fidelity wireframes into polished user interface screens by incorporating modern color palettes, typography, icons, imagery, and interactive UI components. The application enables users to discover restaurants, browse food categories, explore menus, manage their cart, complete secure checkouts, and track orders through a clean and intuitive interface.

**Tools Used**
  **1. Figma** – UI Design and Prototyping
  **2. Microsoft Word** – Project Documentation
  **3. PDF** – Final Design Report

**Features**
  1. User Login and Registration
  2. Restaurant Discovery
  3. Food Category Browsing
  4. Menu Exploration
  5. Add-to-Cart Functionality
  6. Order Summary and Checkout
  7. Order Confirmation and Tracking

**Design Highlights**
  1. Clean and Modern Mobile Interface
  2. Consistent Visual Hierarchy
  3. User-Friendly Navigation
  4. High-Quality Food Imagery
  5. Organized Information Architecture
  6. Simplified Ordering Workflow
  7. Responsive and Accessible Design Principles

**Screens Included**
  1. Login & Authentication
  2. Home Dashboard
  3. Restaurant Menu
  4. Cart Summary
  5. Checkout
  6. Order Confirmation

**Outcome:** The final design delivers an intuitive and visually appealing food delivery experience focused on usability, accessibility, and efficient user interaction. The project successfully demonstrates the complete user journey from user authentication to successful order placement while maintaining consistency in design and navigation.
